# VUT 
Vue Utility Tool
> Vue.js 的移动开发UI组件 - 专注于解决常见开发场景的问题 

####  组件的核心思想：
- 样式(外观直接可以看见的)
- 功能(改变自身状态)
- 动作/交互(与其他外部交流))

####  基础组件
- [ ] 按钮
- [ ] 图标
